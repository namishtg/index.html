Privacy Policy for Industrial47 WA Manager

Last Updated: June 23, 2025

Data Collection

This extension does not collect, transmit, or store any personal data on external servers. All functionality operates locally on your device.

Local Storage

The extension stores the following data locally on your device using Chrome's storage API:

  • Custom chat tags and organizational preferences
  • Reminder schedules and nudge settings
  • Extension settings and configurations
  • Optional Streak CRM integration settings (API key stored locally when provided by user)

WhatsApp Web Integration

The extension only interacts with WhatsApp Web's interface elements (chat names, visual components) and does not access, read, or store message content.

Third-Party Services

Optional Streak CRM integration:

  • Requires user-provided API key stored locally
  • Only syncs contact names and basic metadata when explicitly requested by user
  • No data is shared without explicit user action

Data Sharing

No user data is shared with third parties, sold, or transmitted to external servers.
